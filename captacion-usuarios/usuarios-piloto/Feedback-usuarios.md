# Feedback de usuarios

## Problemas de uso principales

- No se guarda sesión al recargar.
- Uso bidireccional de cuentas (inquilino y propietario).
- Buscar solución para inmobiliarias (cómo gestionar cuando el piso pertenece a un propietario y ciertas gestiones las realiza la inmobiliaria).
- Meter 2FA por seguridad.
- Contenedores con diferentes tamaños en diferentes pantallas: unos tienen como un "overlay" tipo móvil, otros no.
- Menús desordenados.

## Publicación de inmuebles

- Poner para elegir ciudad, introducir manualmente barrio y poner botón de "otros" en gastos incluidos.
- Falta mucha info de los pisos.
- Se duplica al darle varias veces a publicar (al darle la primera debería desactivarse el botón o similar).
- No queda claro si es piso entero, habitación o si se puede ambas.
- Si la foto del inmueble no es válida, lanzar mensaje de error.

## Match

- La info básica de usuario debería ser obligatoria (al crear la cuenta, por ejemplo), porque si no no se puede elegir bien si aceptar o no una solicitud.
- Explicar cómo funciona el sistema de match (quién lo envía primero, quién lo acepta, qué se hace luego, cómo se añade a la vivienda, etc.).
- Poder buscar pisos sin match, enviar un piso a alguien o invitar a alguien a la vivienda para dar más flexibilidad.
- (Propietario) En "Mis matchs" que salga la lista de usuarios según cada inmueble y un botón de aceptar para introducirlo en el piso; además, que tenga un menú aparte para solicitud de invitación al piso.

## Otros

- Favoritos no funciona (y no tiene mucho sentido que exista).
- Limitaciones: fecha de nacimiento, disponibilidad de un piso y longitud de caracteres (este último puede petar la app).
- Ver detalles o fotos de manera más intuitiva, metiendo botones o explicando que hay que deslizar hacia arriba.

## Sugerencias

- Aclarar el tema de los códigos de error. Si metemos algo de Swagger/OpenAPI, puede gustar y además seguir directrices. Se puede arreglar haciendo un modal de frontend (no mensaje del navegador) y aclarando códigos de error.
- Personalmente, lo de poder editar siempre no me gusta (opinión personal): no soy consciente de cuándo tengo que guardar cambios o cuándo estoy editando.
- Sería interesante borrar solicitudes.
- Servicio PERSONA para autentificación de usuarios. Se puede hablar con ellos para una prueba gratis (60 días en preproducción y, si se acuerda, una prueba real). Es un componente muy sencillo.
- DigitalOcean: 200 dólares gratis.
